    How to Play

    To start the game, click on the "Play the Game!" link above, then click on "New Game" once the game has loaded.

    Objective: Guide the lander to the platform on the planet while avoiding obstacles. The HUD will warn you if you are approaching at an unsafe speed. If the landing is successful, the game will pause and then move back to the title screen.

    Controls: Up Arrow Key: Thrust Left/Right Arrow Keys: Spin

    About

    This game is a project for CMPM 20 at UCSC. It has been created in HTML 5 and Javascript using the Phaser library.

    This game is currently still in development and is not finished yet. Do not treat it as a finished product.

    DISCLAIMER: The game is temporarily being hosted through Google Drive. The load times are very long at the moment to load the audio. Please allow time for it to load!
